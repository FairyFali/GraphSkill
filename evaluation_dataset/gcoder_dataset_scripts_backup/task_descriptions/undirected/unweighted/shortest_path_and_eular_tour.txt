Define a **hybrid structural score** for an unweighted, undirected graph $(G)$ with two distinguished nodes $(s)$ and $(t)$.

* If $(G)$ supports a trail that uses every edge exactly once, the score is given by the the maximum shortest-path distance $\operatorname{diam}(G) $ between any pair of nodes in the graph of $(G)$.
* If $(G)$ does not support, the score is defined instead as
  $$
  \max{C(s), C(t)},
  $$
  where $(C(\cdot))$ denotes the local clustering coefficient of a node.

Given $(G)$, $(s)$, and $(t)$, compute this hybrid structural score and return its value.
