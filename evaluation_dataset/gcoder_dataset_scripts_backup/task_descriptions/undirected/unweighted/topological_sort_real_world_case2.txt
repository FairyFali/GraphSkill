Imagine a spreadsheet where many cells contain formulas that reference other cells. You need to choose an order to recalculate the cells so that every cell is computed only after all the cells it depends on have been updated. If there’s a circular reference (e.g., A1 depends on B1, and B1 depends on A1, directly or indirectly), then no valid recalculation order exists and the spreadsheet should report an error.
