You are given an undirected graph
$
G = (V, E),
$
represented by a list of edges.

Let $ \mathcal{C}(G) = { C_1, C_2, \dots, C_m } $ denote the set of of nodes in a graph such that any two nodes in the subset can be reached from one another by following edges of $ G $. Define the largest number $ H $ of edges required to travel between any pair of nodes when taking the shortest possible route between them of a connected graph  as
$$
\operatorname{diam}(H) =
\begin{cases}
\max\limits_{u,v \in V(H)} d_H(u,v), & |V(H)| > 1, 
\\\\\
0, & |V(H)| = 1,
\end{cases}
$$
where $ d_H(u,v) $ is the length (number of edges) of the shortest path between nodes $ u $ and $ v $ in $ H $.

Define the **effective diameter** of $ G $ as
$$
D(G) =
\begin{cases}
\operatorname{diam}(G), & \text{if } |\mathcal{C}(G)| = 1, \\\\
\operatorname{diam}(H^\ast), & \text{if } |\mathcal{C}(G)| > 1,
\end{cases}
$$
where
$$
H^\ast = \arg\max_{C \in \mathcal{C}(G)} |V(C)|
$$
is a connected component of $ G $ with the maximum number of nodes (ties are resolved arbitrarily).

If $ V = \varnothing $, no value is returned.

---

**Output:**
Return the value $ D(G) $ as defined above.
