Sports league fairness check: Model the season’s schedule as a directed, unweighted graph where each node is a team and each game is represented by a directed edge from the home team to the away team. Using this graph, verify that every team has the same total degree (sum of in-degree and out-degree), meaning each team participates in the same number of games overall. If all teams have identical total degrees, the schedule is balanced; otherwise, flag teams whose total number of games (incoming plus outgoing edges) differs from the rest.
