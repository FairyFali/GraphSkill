A university curriculum planner models course prerequisites as a directed graph G = (V, E), where each node is a course and each edge (u, v) indicates that course u must be completed before course v. To produce a valid sequence of courses a student can take, your task is to compute a topological ordering of this prerequisite graph. This ensures all prerequisites are taken before dependent courses. If no such ordering exists, the curriculum contains a circular prerequisite that must be resolved before a valid study plan can be created.
