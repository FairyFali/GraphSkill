You are given an undirected, unweighted graph as an edge list and an integer k. The goal is to find nodes that act as strong connectors between different regions of the network.

For each node v, let Γ(v) be the set of its neighbors and k_v = |Γ(v)|. Define the local neighborhood interconnection ratio R(v) as 2E_v / (k_v (k_v − 1)) if k_v ≥ 2 and 0 otherwise, where E_v counts edges whose endpoints both lie in Γ(v). Let d(v, u) be the minimum number of edges on any path from v to a reachable node u ≠ v, and let d̄(v) be the average of d(v, u) over all such reachable nodes; if none exist (v is isolated), set d̄(v) = ∞. The bridge-hub score is S(v) = (1 − R(v)) / d̄(v), with S(v) = 0 when d̄(v) is ∞ or 0. If multiple nodes share the same score, break ties by node identifier in ascending order.

Return the top k nodes with the highest bridge-hub scores, ordered by descending score and then by node identifier, and include each node together with its score.
