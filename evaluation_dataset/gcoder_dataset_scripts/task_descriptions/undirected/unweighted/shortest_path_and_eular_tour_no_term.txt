Compute a hybrid structural score for an undirected, unweighted graph given by an edge list and two distinguished nodes s and t. If an Eulerian trail exists, use the maximum shortest-path distance between any two nodes in the subgraph induced by nodes with positive degree (distance is measured in edges along a shortest path); if there are no such nodes, take this value as 0. Otherwise, use the larger of two local neighbor-interconnection ratios at s and t, where for a node v this ratio is 0 when deg(v) < 2, and otherwise equals the number of edges among v’s neighbors divided by deg(v)·(deg(v)−1)/2. Return the resulting score as a single number.
