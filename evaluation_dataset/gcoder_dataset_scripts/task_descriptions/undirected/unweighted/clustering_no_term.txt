You are given an undirected, unweighted graph represented by an edge list, where each edge is a pair of nodes (u, v) indicating a bidirectional connection between u and v. Given a node (node), compute its local neighbor-connection ratio: the number of edges that exist between the neighbors of node divided by the total possible number of edges between those neighbors, i.e., m / (k*(k-1)/2), where k is the degree of node and m is the number of edges among its neighbors. If k < 2, return 0.0. You should only return the value as a float.
