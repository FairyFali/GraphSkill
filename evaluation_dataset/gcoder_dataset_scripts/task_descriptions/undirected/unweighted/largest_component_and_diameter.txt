You are given an undirected, unweighted graph represented by an edge list, where each edge is a pair of nodes indicating a bidirectional connection. The goal is to compute the graph’s effective diameter based on shortest-path distances.

The node set consists of all unique endpoints appearing in the edge list. A connected component is a maximal subset of nodes in which every pair of nodes is connected by some path of edges. The shortest-path distance between two nodes is the minimum number of edges on any path connecting them within the same component. For any connected graph H, its diameter is the maximum shortest-path distance over all node pairs if H has at least two nodes, and 0 if H has exactly one node. The effective diameter of the input graph G is defined as follows: if G is connected, it is the diameter of G; otherwise, select any connected component with the largest number of nodes (ties broken arbitrarily) and take its diameter. If the graph has no nodes, no value is returned.

Return the effective diameter of the graph as a nonnegative integer, or return no value if the graph has no nodes.
