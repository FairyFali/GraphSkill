You are given an undirected, unweighted graph via an edge list, where each pair denotes a two-way connection. The vertex set is the collection of all distinct endpoints present in the edges. A connected component is a largest possible set of vertices where every two vertices are linked by some path. The shortest-path distance between two vertices is the minimum number of edges on any path joining them within the same component. For any connected graph H, its diameter is the maximum of these shortest-path distances over all vertex pairs when H has at least two vertices, and 0 when H has exactly one vertex. The effective diameter of the input graph G is defined as: if G is connected, it equals the diameter of G; otherwise, choose any component with the greatest number of vertices (breaking ties arbitrarily) and use its diameter. If the graph contains no vertices, no value is produced.

Return the effective diameter as a nonnegative integer, or return no value if the graph has no vertices.
