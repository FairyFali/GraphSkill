You are given an undirected, unweighted graph as an edge list, where each pair (u, v) represents a bidirectional link between u and v. For a specified node, compute its local clustering coefficient: the number of edges present among that node’s neighbors divided by the total number of possible edges between those neighbors, i.e., m / (k*(k-1)/2), where k is the degree of the node and m is the count of edges between its neighbors. If k < 2, return 0.0. Only output the coefficient as a float.
