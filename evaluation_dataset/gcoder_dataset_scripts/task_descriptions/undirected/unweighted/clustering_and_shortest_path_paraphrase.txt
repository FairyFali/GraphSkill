You are given an undirected, unweighted graph specified by an edge list and an integer k. The aim is to find vertices that serve as effective connectors between different regions of the network.

For each vertex v, let Γ(v) be its neighbor set and k_v = |Γ(v)|. Define the local neighborhood interconnection (local clustering coefficient) as C(v) = 2E_v / (k_v (k_v − 1)) when k_v ≥ 2, and 0 otherwise, where E_v is the number of edges whose endpoints both lie in Γ(v). For any reachable node u ≠ v, let d(v, u) be the unweighted shortest-path distance (in hops). Let d̄(v) be the average of d(v, u) over all such reachable u; if no other nodes are reachable from v (i.e., v is isolated), set d̄(v) = ∞. Define the bridge-hub score S(v) = (1 − C(v)) / d̄(v), with S(v) = 0 when d̄(v) is ∞ or 0. Rank nodes by S(v) in descending order, breaking ties by increasing node identifier. Return the top k nodes with their scores.
