You are given an undirected, weighted graph represented by an edge list, where each edge is a triplet (u, v, c) indicating a bidirectional connection between u and v with positive capacity c. Given a source node _snode and a sink node _tnode, compute the maximum flow from _snode to _tnode (treat each undirected edge as two directed edges with the same capacity). The maximum flow problem seeks the largest feasible flow from a source to a sink under capacity constraints. Return only the flow value as a float.
