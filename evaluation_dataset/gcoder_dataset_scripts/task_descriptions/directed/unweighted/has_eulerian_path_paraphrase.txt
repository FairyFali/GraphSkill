You are given a directed, unweighted graph described by an edge list, where each edge is a pair (u, v) meaning a one-way edge from u to v. Implement a function has_eulerian_path() that returns true if the graph has an Eulerian path and false otherwise. A directed graph has an Eulerian path exactly when all vertices with nonzero degree lie in one weakly connected component and, in addition, either (a) every vertex has equal in-degree and out-degree, or (b) there is exactly one vertex whose out-degree is in-degree + 1 and exactly one vertex whose in-degree is out-degree + 1, with all remaining vertices having matching in-degree and out-degree. The function should return a boolean.
