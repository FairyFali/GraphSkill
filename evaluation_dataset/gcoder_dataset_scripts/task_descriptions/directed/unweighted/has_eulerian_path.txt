You are given a directed, unweighted graph represented by an edge list, where each edge is an ordered pair (u, v) indicating a one-way connection from u to v. Implement has_eulerian_path() that returns a boolean indicating whether the graph contains an Eulerian path. A directed graph has an Eulerian path if and only if all vertices with nonzero degree belong to a single weakly connected component and either (a) every vertex has in-degree equal to out-degree, or (b) exactly one vertex has out-degree equal to in-degree + 1 and exactly one vertex has in-degree equal to out-degree + 1, with all other vertices having equal in-degree and out-degree. Return a boolean value.
