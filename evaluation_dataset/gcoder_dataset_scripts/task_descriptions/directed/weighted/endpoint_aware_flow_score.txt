Compute an endpoint-aware flow score for a weighted directed network between two specified endpoints, _snode and _tnode. This score multiplies the maximum flow value by a factor based on the local clustering of the endpoints in the underlying undirected projection.

The input is a directed graph with finite, nonnegative real capacities on edges, represented by triples (u, v, w) where w is the capacity of edge u→v; if multiple triples share the same ordered pair (u, v), their capacities are summed. Let F(_snode, _tnode) be the maximum flow value from _snode to _tnode respecting these capacities. Construct an undirected simple graph on the same node set by replacing every directed edge (u, v) with a single undirected edge {u, v} (if both directions exist, keep only one; ignore weights). For any node x, let C(x) be its local clustering coefficient in this undirected graph, defined as the fraction of unordered neighbor pairs of x that are connected by an edge (and 0 if x has fewer than two neighbors). Define S = F(_snode, _tnode) * (1 + (C(_snode) + C(_tnode)) / 2).

Return S as a real-valued number.
