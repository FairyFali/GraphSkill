**This task is to determine a valid linear ordering of the nodes in a directed graph ( G = (V, E) ).** The goal is to produce a sequence in which every node appears before all nodes that depend on it—formally, for each directed edge ((u, v) \in E), (u) must come earlier than (v). Such an ordering can be constructed only if the graph contains no directed cycles, meaning the graph must be a directed acyclic graph (DAG).
