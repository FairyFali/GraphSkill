Real‑world paraphrase: Think of a warehouse’s one‑way conveyor system where junctions are hubs and conveyor segments carry packages in a single direction. You want to send a single test parcel that rides every conveyor segment exactly once without lifting it off the system. This is possible only if, ignoring the one‑way rules, all hubs belong to one connected network, and the flow at hubs is balanced: either every hub has the same number of incoming and outgoing belts (you can start and end at the same hub), or exactly two hubs are imbalanced—one has one extra outgoing belt (start) and one has one extra incoming belt (finish).
