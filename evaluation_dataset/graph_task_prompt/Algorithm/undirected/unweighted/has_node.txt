**Node existence check for a graph:**
Given a graph (G = (V, E)) and a node (v), the task is to determine whether (v) is present in the node set (V) of the graph. This is done by checking whether the identifier of (v) appears among the nodes defined or implied by the graph structure.

---

### Example 1: An undirected graph

Graph: nodes ({A, B, C, D}), undirected edges: ({A,B}, {B,C}, {C,D})
Task: check whether node (A) exists in the graph

**Step 1:** Enumerate the nodes of the graph.
From the edge list, the nodes are ({A, B, C, D}).

**Step 2:** Compare the given node with the node set.
Check whether node (A) appears in the set of nodes.

**Step 3:** Conclude the result.
Node (A) is present in the graph, so the answer is **True**.
