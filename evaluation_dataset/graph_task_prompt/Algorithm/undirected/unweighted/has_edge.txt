**Edge existence check for a graph:**
Given a graph (G = (V, E)) and two nodes (u) and (v), the task is to determine whether there is an edge directly connecting (u) and (v). In an undirected graph, this means checking whether the unordered pair ({u,v}) is in the edge set; in a directed graph, this means checking whether the ordered pair ((u,v)) is in the edge set.

---

### Example 1: An undirected graph

Graph: nodes ({A, B, C, D}), undirected edges: ({A,B}, {B,C}, {C,D})
Task: check whether there is an edge between nodes (A) and (C)

**Step 1:** Identify the neighbors of node (A).
From the edge list, the neighbors of (A) are ({B}).

**Step 2:** Check whether node (C) appears in the neighbor set of (A).
Node (C) is not in ({B}).

**Step 3:** Conclude the result.
Since (A) and (C) are not directly connected by an edge, the answer is **False**.
