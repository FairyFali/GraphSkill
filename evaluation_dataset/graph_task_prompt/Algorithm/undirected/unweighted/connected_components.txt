Breadth First Search (BFS) / Depth First Search (DFS) component expansion; The idea is to repeatedly pick an unvisited node, run BFS (or DFS) to collect all nodes reachable from it as one connected component, mark them visited, and repeat until every node belongs to some component.

Example 1: Two connected components
Graph: nodes {1, 2, 3, 4, 5}, edges: (1,2), (2,3), (4,5)
Step 1: Initialize visited = empty, components = empty. Pick an unvisited node, for example node 1. Start BFS from 1.
Step 2: Visit 1, mark 1 visited, queue = [1].
Step 3: Dequeue 1, its neighbor is 2. Mark 2 visited and enqueue it. Now queue = [2].
Step 4: Dequeue 2, its neighbors are 1 and 3. Node 1 is already visited; mark 3 visited and enqueue it. Now queue = [3].
Step 5: Dequeue 3, its neighbor is 2 which is already visited. Queue is empty, so the first component is {1,2,3}. Add it to components.
Step 6: Find the next unvisited node: node 4. Start BFS from 4.
Step 7: Visit 4, mark 4 visited, queue = [4]. Dequeue 4, visit neighbor 5, mark 5 visited, enqueue 5.
Step 8: Dequeue 5, its neighbor 4 is already visited. Queue is empty, so the second component is {4,5}. Add it to components.
Step 9: All nodes are now visited, so the connected components are [{1,2,3}, {4,5}].
