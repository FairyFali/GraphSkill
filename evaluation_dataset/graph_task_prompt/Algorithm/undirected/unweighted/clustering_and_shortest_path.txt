Local clustering coefficient computation for a directed, unweighted graph: For a given node v, we first take all of its out-neighbors (all nodes u such that there is a directed edge (v,u)). Among this neighbor set, we count how many directed edges actually exist between those neighbors, and we also compute how many directed edges could possibly exist in total between them. We then define the clustering coefficient of v as (number of directed edges between neighbors of v) divided by (maximum possible number of directed edges between neighbors of v). If v has fewer than two neighbors, its clustering coefficient is defined as zero.

Example 1: A directed triangle where the center node has high clustering
Graph: nodes {A, B, C}, directed edges: (A,B), (A,C), (B,C)
Task: compute clustering coefficient of node A

Step 1: List the out-neighbors of A. From edges (A,B) and (A,C), the neighbors are {B, C}. There are 2 neighbors.

Step 2: Count directed edges among neighbors. Between B and C there is 1 directed edge: (B,C). So the number of existing directed edges between neighbors is 1.

Step 3: Compute possible directed edges among neighbors. With 2 neighbors, the maximum number of directed edges between them is 2, corresponding to the ordered pairs (B,C) and (C,B).

Step 4: Clustering coefficient of A is existing directed edges between neighbors divided by possible directed edges between neighbors, which is 1 divided by 2, giving 0.5.

Dijkstra's Algorithm; The idea is to use a priority queue to repeatedly select the node with the smallest known distance from the source, relax all of its edges by updating neighbor distances if a shorter path is found, and continue until all reachable nodes have been processed or the target node is reached. The final recorded distance for the target node gives the minimum total weight of the shortest path.

Example 1: A simple linear graph
Graph: nodes {A, B, C}, edges: (A,B) weight 2, (B,C) weight 3
Task: shortest path from A to C
Step 1: Initialize distances: dist(A)=0, dist(B)=infinity, dist(C)=infinity.
Step 2: Select A. Relax A to B: dist(B)=2.
Step 3: Select B (next smallest distance). Relax B to C: dist(C)=2+3=5.
Step 4: Select C. No more neighbors to update.
Conclusion: Shortest path from A to C is weight 5, with path A to B to C.
