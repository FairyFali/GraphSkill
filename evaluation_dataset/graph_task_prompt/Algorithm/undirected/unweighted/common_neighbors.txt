Neighbor set intersection; The idea is to list the neighbors of node u and the neighbors of node v, then compute the intersection of these two neighbor sets to get the common neighbors.

Example 1: Two nodes with shared friends
Graph: nodes {1, 2, 3, 4, 5}, edges: (1,2), (1,3), (1,4), (2,3), (2,5)
Goal: find the common neighbors of nodes 1 and 2
Step 1: List the neighbors of node 1. Neighbors(1) = {2, 3, 4}.
Step 2: List the neighbors of node 2. Neighbors(2) = {1, 3, 5}.
Step 3: Compute the intersection of the two sets: {2, 3, 4} ∩ {1, 3, 5} = {3}.
Step 4: The common neighbors of nodes 1 and 2 are {3}, so the number of common neighbors is 1.
