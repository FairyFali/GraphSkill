Breadth First Search (BFS) all-pairs eccentricity; The idea is to run BFS from each node to get its shortest-path distances to all other nodes, take the farthest distance from that node (its eccentricity), and then take the maximum eccentricity over all nodes as the graph diameter.

Example 1: A simple connected chain
Graph: nodes {1, 2, 3, 4}, edges: (1,2), (2,3), (3,4)
Step 1: Run BFS starting from node 1 and compute shortest distances: d(1,1)=0, d(1,2)=1, d(1,3)=2, d(1,4)=3. The farthest distance from 1 is 3.
Step 2: Run BFS starting from node 2: d(2,1)=1, d(2,2)=0, d(2,3)=1, d(2,4)=2. The farthest distance from 2 is 2.
Step 3: Run BFS starting from node 3: d(3,1)=2, d(3,2)=1, d(3,3)=0, d(3,4)=1. The farthest distance from 3 is 2.
Step 4: Run BFS starting from node 4: d(4,1)=3, d(4,2)=2, d(4,3)=1, d(4,4)=0. The farthest distance from 4 is 3.
Step 5: Take the maximum of these farthest distances: max(3, 2, 2, 3) = 3, so the diameter of the graph is 3.
