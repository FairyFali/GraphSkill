### Example 1: A directed graph with a cycle

Graph: nodes {A, B, C, D}, directed edges: (A,B), (B,C), (C,A), (C,D)
Task: check whether the graph is cyclic

**Step 1:** Initialize all nodes as unvisited, and initialize an empty recursion stack.

**Step 2:** Start DFS from node A. Mark A as visited and add A to the recursion stack.

**Step 3:** From A, visit node B. Mark B as visited and add B to the recursion stack.

**Step 4:** From B, visit node C. Mark C as visited and add C to the recursion stack.

**Step 5:** From C, examine its outgoing edges. One edge leads to A, which is already in the recursion stack.

**Step 6:** Encountering a node already in the recursion stack indicates a cycle. The graph is cyclic.

**Step 7:** If DFS completes for all nodes without encountering such a back edge, then the graph is acyclic.
