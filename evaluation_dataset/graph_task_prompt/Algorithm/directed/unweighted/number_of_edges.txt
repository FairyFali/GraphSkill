Given a graph (G = (V, E)), the task is to determine the total number of edges in the graph. Each edge in the edge set (E) is counted once, regardless of how many nodes it connects or whether it is directed or undirected.
Graph: nodes ({A, B, C, D}), directed edges: ((A,B), (B,C), (C,A), (D,A))
Task: count the number of edges in the graph

**Step 1:** Identify all directed edges in the graph.
From the edge list, the directed edges are ((A,B), (B,C), (C,A), (D,A)).

**Step 2:** Count the directed edges.
