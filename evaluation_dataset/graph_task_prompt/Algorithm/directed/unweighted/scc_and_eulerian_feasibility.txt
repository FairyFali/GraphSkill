Hierholzer's Algorithm; The idea is to check whether the graph has an Euler path by verifying two conditions: first, the graph must be connected when ignoring isolated nodes; second, the number of vertices with odd degree must be either zero or two. If both conditions hold, an Euler path exists; otherwise, it does not.

Example 1: A simple path that has an Euler path
Graph: nodes {1, 2, 3}, edges: (1,2), (2,3)
Step 1: Compute degrees. Node 1 has degree 1, node 2 has degree 2, node 3 has degree 1.
Step 2: Count odd-degree vertices. Nodes 1 and 3 have odd degree, giving exactly two odd-degree vertices.
Step 3: Check connectedness. All nodes are connected through edges.
Conclusion: Conditions are satisfied, so an Euler path exists.

Kosaraju’s algorithm (two-pass DFS); The idea is to run a DFS to get nodes in decreasing finish time order, then run DFS on the reversed graph in that order, where each DFS tree forms one strongly connected component.

Example 1: Two SCCs in a directed graph
Graph: nodes {1, 2, 3, 4, 5}, directed edges: (1,2), (2,3), (3,1), (3,4), (4,5), (5,4)
Step 1: Run a DFS on the original graph and record each node’s finish time (when you are done exploring it). One valid finish order (last finished first) is: [1, 2, 3, 4, 5] as a finish-time stack [5, 4, 3, 2, 1] (top is left).
Step 2: Reverse all edges to form the transpose graph. The reversed edges are: (2,1), (3,2), (1,3), (4,3), (5,4), (4,5).
Step 3: Pop the top of the finish-time stack, start DFS from node 5 in the reversed graph. From 5 you can reach 4, and from 4 you can reach 5, so you collect the SCC {4,5}. Mark {4,5} as visited in the reversed graph.
Step 4: Continue popping: next is 4, but it is already visited, so skip it.
Step 5: Next is 3, start DFS from node 3 in the reversed graph. From 3 you can reach 2 (via 3->2), from 2 you can reach 1 (via 2->1), and from 1 you can reach 3 (via 1->3), so you collect the SCC {1,2,3}. Mark {1,2,3} as visited.
Step 6: Pop the remaining nodes 2 and 1, both already visited, so you are done. The strongly connected components are {4,5} and {1,2,3}.
